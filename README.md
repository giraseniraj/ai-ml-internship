## Assignment Overview

This program analyzes the scores of students using **Python** and **NumPy**.

The program performs the following tasks:

- Stores student marks in a Python list.
- Assigns grades based on marks.
- Displays each student's score with the corresponding grade.
- Converts the list into a NumPy array.
- Calculates:
  - Mean
  - Median
  - Maximum Score
  - Minimum Score
  - Standard Deviation
- Counts how many students scored above the class average.
- Prints only the scores that received an **A Grade (>=85)** using NumPy boolean indexing.
- **Bonus:** Adds five more student scores and finds the grade category with the highest number of students.

## Technologies Used

- Python 3
- NumPy

## Grade Criteria

| Marks | Grade |
|--------|-------|

| 70 - 84 | B |
